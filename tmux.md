# tmux cheatsheet

## General

* **C-Space** - send prefix

## Select pane

* **h** - select left pane
* **j** - select down pane
* **k** - select up pane
* **l** - select right pane

## Split window/pane

* **v** - split window/pane vertically
* **s** - split window/pane horizontally

## Resize pane

* **<** - extend pane to left
* **>** - extend pane to right
* **+** - extend pane to up
* **-** - extend pane to down
